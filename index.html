DarkOs

Is A Linux Distribution Based On Arch Linux, Designed To Create More Enjoyable User Experience With Ease Of Use . DarkOs Comes With 3 Editions (One Hell - Soopertrack - Schmedding), Each Edition Has It's Own Window Manager And Package List.

DarkOs "One Hell

Edition Comes With {Xmonad,Herbstluft,Awesome} Window Manager And Minimal Amount Of Packages And It Splits To Two Seperate Releases, One Is Lightweight And Doesn't Include Much Packages,The Second Comes With Essential Packages With All Apps And Utilities.

DarkOs "Soopertrack"

Edition Comes With No Desktop Or Window Manager But Uses XFCE As Delivery Environment To Provide You With The Installer And It'll Be Removed Once The Installation Is Finished,It Also Comes With No Essential Packages Such As (Browser , Text Editor , Terminal ...etc) So You'll Get To Choose What You Want While Installing.

DarkOs "Schmedding"

Comes With No Desktop Or Window Manager Only A Tty Shell And You Gonna Have To Do The Installation Manually On Your Own But It Provides An Optional Ncurses Installer If You're Lazy To Do The Work .
